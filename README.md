# BUDA-535-Business-Data-Mining

Spring 2026 | West Virginia University

This course bridges data mining techniques with real-world business applications. Topics cover supervised and unsupervised learning, neural networks, and hands-on Python programming.
Modules

- Module 1 – Decision trees and logistic regression
- Module 2 – Model evaluation and support vector machines
- Module 3 – Bayes methods and ensemble methods
- Module 4 – Clustering: K-means, hierarchical, DBSCAN
- Module 5 – Mixture models, PCA, and association analysis
- Module 6 – Deep learning, CNNs, and transfer learning
- Module 7 – RNNs, word embeddings, and sentiment analysis
- Module 8 – Group Project
